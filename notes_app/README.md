# 📝 Notes App

A simple and modern Notes Application built using **React.js**, **Tailwind CSS**, and **JavaScript**.  
This app allows users to create, manage, and organize notes efficiently with a clean and responsive UI.

---

## 📌 Features

- ✍️ Create new notes
- 💾 Data stored locally (LocalStorage)
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS
- 📜 JavaScript (ES6+)
- 💾 Local Storage
